# Post-v1 features

Features planned for releases after v1. These are **not** tracked as v1 GitHub issues.

## Search command

Like `books-cli`: `cards search` with repeatable `--term` flags; OR-match across card front/back and deck names; `--json` for agents.

## CSV / JSON import and export

Bulk load and dump decks and cards for backup and portability.

## Shuffle deck

Reshuffle the entire deck queue order (not just the current study batch). Useful for breaking fixed patterns; still queue-position based, no date scheduling.

---

**Explicitly excluded** from future scope: tags, stale bump, books-cli integration, markdown rendering, Anki import/export, media, date-based scheduling, sync, cloze deletions, web UI, archive.
